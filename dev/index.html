Geotherm

Documentation for Geotherm.

The code is hosted on GitHub, with some continuous integration services to test its validity. For corresponding paper, please have a look at Ref. 1.

This repository is created and maintained by @singularitti. You are very welcome to contribute.

Compatibility

  • Julia version: v1.0.0 to v1.6.2
  • Dependencies:
    • BisectPy.jl v1.0.0 and above
    • DimensionalData.jl v0.4.0 and above
  • OS: macOS, Linux, Windows, and FreeBSD
  • Architecture: x86, x64, ARM
